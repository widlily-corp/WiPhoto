# WiPhoto v1.5.1 - Changelog

## 🎉 Новые возможности

### 🎬 Поддержка видео
- Добавлена полная поддержка видеофайлов
- Встроенный видеоплеер с элементами управления
- Поддерживаемые форматы: MP4, AVI, MKV, MOV, WMV, FLV, WebM, M4V, MPEG, 3GP и другие
- Автоматическое создание миниатюр из первого кадра видео
- Двойной клик на видео открывает плеер с возможностью воспроизведения

### 🖼️ Расширенная поддержка форматов изображений
Добавлена поддержка всех популярных форматов:
- **Современные форматы**: HEIC, HEIF, AVIF, WebP
- **RAW форматы**: CR3, ORF, PEF, RAF, SRW, X3F, RW2 и 20+ других
- **Дополнительные форматы**: JPEG 2000 (JP2, J2K), ICO, PPM, PGM, PBM

### 🎨 Улучшенное удаление объектов
- Модернизирована модель удаления объектов
- Увеличено максимальное разрешение обработки (до 2048px)
- Добавлено сглаживание границ для плавного перехода
- Fallback на улучшенный OpenCV метод при недоступности LaMa
- Комбинация алгоритмов Telea и Navier-Stokes для лучшего качества
- Постобработка с bilateral фильтром

### 🔍 Масштабирование миниатюр
- Добавлена возможность изменения размера миниатюр с помощью Ctrl + колесо мыши
- 7 предустановленных размеров: от 128px до 512px
- Плавное изменение размера в реальном времени
- Работает во всех режимах просмотра

## 📋 Использование новых функций

### Работа с видео
1. Откройте папку с видео и изображениями
2. Видеофайлы отображаются с миниатюрами в галерее
3. Двойной клик по видео открывает встроенный плеер
4. Используйте элементы управления для воспроизведения

### Изменение размера миниатюр
1. Наведите курсор на область с миниатюрами
2. Зажмите `Ctrl` и вращайте колесо мыши
3. Вверх - увеличить, вниз - уменьшить
4. Доступны размеры: 128, 160, 200, 256, 320, 400, 512 пикселей

### Улучшенное удаление объектов
1. Откройте изображение в редакторе
2. Выберите инструмент "Умное ретуширование"
3. Закрасьте объект для удаления
4. Нажмите "Применить"
5. Наслаждайтесь улучшенным качеством

### Из исходного кода
```bash
git pull origin main
pip install -r requirements.txt
```

### Скомпилированная версия
1. Скачайте новую версию
2. Распакуйте в новую папку
3. Перенесите настройки из `~/.wiphoto/` (опционально)

## 📝 Примечания

- Распознавание лиц использует Haar Cascades (встроено)
- Для работы с GPS требуется ExifTool (включен в Windows сборку)
- Видео требует OpenCV с поддержкой FFmpeg (обычно включено)
- AI удаление объектов загружает модель при первом использовании (~100MB)

## 🐛 Известные проблемы

- Некоторые видео кодеки могут не поддерживаться (зависит от FFmpeg)
- HEIC/HEIF требуют дополнительных библиотек на некоторых системах
- Очень большие видео (>4K) могут воспроизводиться с задержкой
