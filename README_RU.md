# WiPhoto

**Профессиональный менеджер и редактор фотографий**

WiPhoto - это мощное приложение для управления и редактирования фотографий, созданное на Python и PyQt6. Программа сочетает продвинутые возможности обработки изображений с интуитивным интерфейсом для эффективной организации, анализа и редактирования фотоколлекций.

## ✨ Возможности

### Управление фотографиями
- **Умное сканирование**: Многопоточное сканирование папок с отображением прогресса
- **Поддержка RAW**: Нативная поддержка форматов `.arw`, `.cr2`, `.nef`, `.dng`, `.raw` через rawpy
- **Поиск дубликатов**: Продвинутый алгоритм перцептивного хеширования для обнаружения похожих изображений
- **Умные коллекции**: Интеллектуальная фильтрация и организация
- **Просмотр метаданных**: Полный просмотр EXIF данных с использованием ExifTool
- **Пакетные операции**: Копирование, перемещение или удаление множества файлов сразу

### Недеструктивное редактирование
- **Предпросмотр в реальном времени**: Мгновенная визуальная обратная связь с регулируемым качеством
- **Система отмены/повтора**: Полное дерево истории для всех операций редактирования
- **Профессиональные инструменты**:
  - **Свет**: Экспозиция, Контраст, Света, Тени, Белые, Черные
  - **Цвет**: Температура, Оттенок, Насыщенность, Яркость
  - **Детали**: Четкость, Резкость
  - **Эффекты**: Виньетка
  - **Трансформация**: Кадрирование, Поворот, Отражение (по горизонтали/вертикали)

### Интерфейс
- **Современный дизайн**: Кастомная темная тема "Liquid Glass" с эффектами glass-morphism
- **Многовкладочный интерфейс**: Галерея, Умные коллекции, Сравнение и вкладки Редактора
- **Сравнение изображений**: Просмотр фотографий рядом для сравнения
- **Горячие клавиши**: Эффективная работа с помощью клавиатурных сокращений
- **Drag & Drop**: Простой импорт файлов

### Совместимость с Linux

WiPhoto может работать на Linux со следующей настройкой:

**Протестированные дистрибутивы:**
- Ubuntu 22.04 LTS и новее
- Debian 11+
- Fedora 36+
- Arch Linux (последняя версия)

**Инструкция по установке для Linux:**

1. Установите системные зависимости:
   ```bash
   # Ubuntu/Debian
   sudo apt install python3-pyqt6 python3-pip libexiftool-perl

   # Fedora
   sudo dnf install python3-qt6 python3-pip perl-Image-ExifTool

   # Arch
   sudo pacman -S python-pyqt6 python-pip perl-image-exiftool
   ```

2. Установите Python зависимости:
   ```bash
   pip install -r requirements.txt
   ```

3. Измените `core/metadata_reader.py`:
   - Замените `exiftool.exe` на `exiftool` (системный ExifTool)
   - Или скачайте Linux версию ExifTool с https://exiftool.org/

**Известные проблемы:**
- Некоторые элементы UI могут выглядеть по-другому в разных окружениях рабочего стола
- Протестировано с GNOME, KDE Plasma и XFCE
- Темная тема лучше всего работает с темными системными темами

**Заметки по производительности:**
- Версия для Linux обычно работает на 10-15% быстрее благодаря лучшей обработке процессов
- Обработка RAW файлов выигрывает от поддержки нативных библиотек

## 🚀 Установка

### Из исходного кода

1. **Клонируйте репозиторий**:
   ```bash
   git clone https://github.com/widlily-corp/WiPhoto.git
   cd WiPhoto
   ```

2. **Создайте виртуальное окружение**:
   ```bash
   python -m venv .venv
   source .venv/bin/activate  # На Windows: .venv\Scripts\activate
   ```

3. **Установите зависимости**:
   ```bash
   pip install -r requirements.txt
   ```

4. **Скачайте ExifTool** (только Windows):
   - Скачайте с [https://exiftool.org/](https://exiftool.org/)
   - Поместите `exiftool.exe` и папку `exiftool_files` в корень проекта

5. **Запустите приложение**:
   ```bash
   python main.py
   ```

### Из готового релиза

1. Скачайте архив для вашей ОС из раздела [Releases](https://github.com/widlily-corp/WiPhoto/releases)
2. Распакуйте архив
3. Запустите:
   - **Windows**: `WiPhoto.exe`
   - **Linux**: `./wiphoto` или `python main.py`

### Сборка исполняемого файла (Windows)

```bash
python setup.py build
```

Исполняемый файл будет создан в папке `build/`.

## 🎯 Использование

### Начало работы

1. **Запустите WiPhoto**: Выполните `python main.py` или используйте собранный исполняемый файл
2. **Выберите папку**: Укажите папку с вашими фотографиями
3. **Дождитесь сканирования**: Приложение проанализирует все изображения и создаст миниатюры
4. **Исследуйте**: Используйте вкладки для навигации между Галереей, Умными коллекциями, Сравнением и Редактором

### Горячие клавиши

| Действие | Клавиша |
|----------|---------|
| Настройки | `Ctrl + ,` |
| Удалить | `Delete` |
| Копировать | `Ctrl + C` |
| Переместить | `Ctrl + X` |
| Поворот | `R` |
| Сравнить | `Ctrl + D` |
| Полный экран | `F11` |
| Быстрый просмотр | `Space` |
| Следующее изображение | `→` |
| Предыдущее изображение | `←` |
| Выделить всё | `Ctrl + A` |
| Снять выделение | `Ctrl + Shift + A` |
| Обновить | `F5` |
| Выход | `Ctrl + Q` |

### Рабочий процесс редактирования

1. Выберите изображение в Галерее
2. Переключитесь на вкладку **Редактор**
3. Настройте параметры с помощью ползунков в панели управления
4. Смотрите предпросмотр изменений в реальном времени
5. Используйте **Сброс** для отмены отдельных инструментов или **Сбросить всё** для полной отмены
6. Экспортируйте результат, когда будете довольны

## 🛠️ Архитектура

WiPhoto следует паттерну Model-View-Controller (MVC):

- **Модели** (`models/`): Структуры данных (например, `ImageInfo`)
- **Представления** (`views/`): Qt компоненты интерфейса
- **Контроллеры** (`controllers/`): Бизнес-логика и координация
- **Ядро** (`core/`): Обработка изображений, сканирование файлов, поиск дубликатов

### Ключевые компоненты

- **Image Processor** (`core/editing/image_processor.py`): Пайплайн недеструктивного редактирования
- **File Scanner** (`core/file_scanner.py`): Многопроцессное сканирование изображений
- **Editing Tools** (`core/editing/tool_*.py`): Отдельные инструменты настройки изображений
- **Metadata Reader** (`core/metadata_reader.py`): Интеграция с ExifTool

## 🔮 Планируемые функции

### Предстоящие улучшения

- **Поддержка геотегов**: Отображение и редактирование GPS метаданных, интеграция с картами
- **Распознавание лиц**: Автоматическое распознавание лиц и организация
- **Распознавание животных**: Определение и тегирование питомцев/дикой природы на фото
- **Сканирование документов**: Автоопределение границ документа, коррекция перспективы, OCR
- **Дополнительные форматы экспорта**: Поддержка большего количества форматов вывода
- **Облачная интеграция**: Возможности резервного копирования и синхронизации
